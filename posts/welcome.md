# Welcome to My Minimalist Blog

This blog is compiled dynamically on a local Python Flask server by loading markdown text records directly from file files.

## Project Vision
- **No JS dependencies**: Beautiful, light, server-rendered views.
- **Markdown powered**: Simply write standard Markdown documents and drop them inside the posts directory.
- **Fast loading**: Maximizes local rendering speeds.

Hope you enjoy this minimalist setup!
